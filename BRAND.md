## Brand Colors
The brand is built on a **gold/amber accent** over a **warm near-black** and warm off-whites.

| Token | Hex | Role |
|-------|-----|------|
| Gold | `#F9C349` | Primary accent / highlights |
| Amber | `#FFB100` | Secondary accent / hover, CTAs |
| Ink | `#1E1810` | Primary text, dark surfaces (warm black) |
| Stone | `#62615C` | Muted / secondary text |
| Slate | `#4B4F58` | Cool dark text / UI lines |
| Cream | `#F8F7F3` | Warm section background |
| Mist | `#F6F7F8` | Cool light background |
| White | `#FFFFFF` | Base background |
| Black | `#000000` | Max contrast |

## Typography
Pulled from the live Google Fonts requests:
- **DM Sans** (300, 400) — body / UI
- **Roboto** (400, 500) — secondary / body
- **Work Sans** (600) — headings / display

Recommended mapping for the rebuild:
- Headings → **Work Sans** (600)
- Body → **DM Sans** (300/400)
